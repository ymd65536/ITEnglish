## Aから始まる英単語

### Archive

アーカイブ
記録、保管庫などの意味、IT業界では長期保存された記録媒体やサービスに使われる。

#### 例文

Now, Azure Archive Storage is available in South Africa North.

#### 翻訳

現在、Azure Archive Storageは南アフリカ共和国北部で利用できます。
